☞ 数组（重点）

   1. 什么是数组？

       数组：一次可以保存多条数据的一个容器。

       数组与变量的区别：

            ✔ 数组可以一次保存多条件数据

            ✔ 变量一次只能保存一条数据

            ✔ 数组属于复杂数据类型（对象）

            ✔ 变量属于简单数据类型


   2. 为什么要学数组？  

       使用场景： 当程序中需要一次保存多条数据，可以使用数组实现。

   3. 如何使用数组？

        ☞ 定义一个数组

          var  自定义数组名称 = [];

        ☞ 给数组赋值：

            1. 在定义数组的时候就可以直接给数组赋值

                例如：  var arry_score = [1, 2, 3, 4, 5, 'abc', '你好世界', true];

                注意：
                    ☞ 在数组中的值，必须以 ',' 隔开

            2. 可以通过索引的方式给数组赋值

                索引：编号，在数组中索引值是从 0 开始的。

                数组名称[索引值] = 值;

                例如： arry[0] = '你好';

            3. 从数组中取值

                 ✔ 通过索引的方式取值

                 数组名称[索引值];


            4. 获取数组中的长度（个数）： 数组名.length;
    

☞ 冒泡排序   

       从大到小排列： 降序排列

       从小到大排列： 升序排列

☞ 数学方法:

     1. Math.floor();   地板函数 : 用来返回一个小于当前数字且与当前数字最接近的一个整数。

     2. Math.ceil();    天花板函数： 用来返回一个大于当前数字且与当前数字最接近的一个整数。

     3. Math.random();  随机数：随机产生的一个数字  [0,1)

     总结：

         ✔ 获取任何值之间的随机整数公式：  Math.floor(Math.random()*(max-min+1)+min);
