> 数组对象
```js

    ☞ 赋值
        
        数组名称.push(number)  : 在数组中的末尾处添加一个值

        数组名称.unshift(number)  : 在数组的最开始添加值

    ☞ 取值

         数组名.pop();  : 从数组的末尾取值，返回一个当前的取值结果

         数组名.shift() : 从数组的开始位置取值，返回一个当前获取值的结果

    ☞ 翻转数组：
         
         数组名.reverse();

    ☞ 拼接字符串

         数组名.join('链接符')   == > 返回的是一个字符串


    ☞ 筛选

        数组名.indexOf()  :  从前向后找
        
        数组名.lastIndexOf() ： 从后向前找


        总结：
             1. 如果在数组中找到对应的值，那么就返回其对应的索引位置

             2. 如果在数组中没有找到对应的值，那么返回结果是 -1;

```

     1. split('-'):  返回是一个数组（必须记）

         字符串.substring(start,end)  ===> 与 slice 一样


         字符串.substr(start, length)  ===> 从start开始截取，一共要截取length个 



     4. 替换

         字符串.replace(a,b);



     5. 拼接字符串

        字符串有一个特点：  不可变
