☞ 学习路径

    HTML（2天） ---->  CSS (7天)  ---->  结合HTML和CSS独立完成一个网站的制作（3天）

☞ 学习目标：

     独立完成网站的制作


☞ HTML学习：

    1. 为甚么要学习HTMl：HTML就是用来制作网站（网页）

    2. 到底什么是网页？

        组成：文字，图片，按钮，输入框，视频，音频...元素组成的。

    3. 制作网页的规则（规范）：

        ☞ 结构标准（HTMl）：制作网页必须使用HTMl

        ☞ 表现标准 （CSS）: 用来美化网页的

        ☞ 行为标准 （javaScript）: 通过js用来实现网页中用户的一些动作（一些网页特效）

    4. 浏览器

        ☞ 什么是浏览器？  用来浏览网页的一款软件

        ☞ 主流浏览器介绍【了解】：

            ✔ IE   谷歌   火狐   苹果浏览器   欧朋浏览器  

            ✔ 浏览器中的核心：
                
               内核：渲染引擎 （将无法识别的图文信息转化可以识别的信息）

            ✔ 主流浏览器中常用的内核：
            
              Trident(IE内核)

              Gecko(firefox)

              webkit(Safari)

              Blink（谷歌）

              Presto(欧朋)

    5. 开始学习HTML： 沟通HTMl制作网页结构。

        ☞  学习HTML的核心： 学习HTML中的各种标签。

        ☞ HTML概念：Hyper(超级)  Text（文本）  Markup（标记）  Language（语言）

              超文本： 就是文本【在网页中用来实现页面跳转的文本】

               标记： 记号（HTMl标记）


        ☞ 先学习HTML的语法结构 : 1. 一个网页必须要有该结构  2. 所有的HTMl标签将来都要写到该结构中
             
             <!doctype html>

             <html>

                <head> </head>

                <body> </body>

             </html>
        
        ☞ 语法结构详解：

              1. <!doctype html>     ----> 告诉浏览器当前文档类型是 HTMl类型

              2. <html></html>    ----> 指的是网页的中的根标签（一个网页中只有一个HTMl）

              3. <head></head>   ---> 
              
                  ✔在网页中表示网页的头部区域（可以在头部区域设置网页的标题）

                  ✔在网页head标签中可以设置网页的样式（css）

                  ✔在网页head标签中可以设置javaScript

                  ✔还可以设置网页的小图标

               4. <body></body>    ----> 代表网页的主体（除去网页的头部区域【地址栏区域】）

                   将来可以把网页中重点显示的文字，图片，按钮标签统统可以放到 body标签中


        ☞HTML标签中写法分类：
             
               1。 单标签（空标签）： 只有开始没有结束

               2。 双标签 ：有开始标签和结束标签


               注意：
                    1。 如果一个标签中希望包含其他内容（文字，图片。。），那么该标签必须是双标签

                    2.  标签只能单独使用，只能使单标签

        ☞HTML标签之间的关系分类

                1. 并列关系： 标签之间不存在嵌套写法

                2. 嵌套关系（父子）： 一个标签中包含其他标签





     1. 先创建一个根目录结构（文件夹）：  打开编译器（文件） ----》 打开文件夹  ---》 选中对应的文件夹

     2. 先使用vscode创建一个HTMl文件，文件后缀名是 .html

        ✔ 新建文件

        ✔ 在html文件中 输入  ! + 回车 （tab）   ---> 生成HTML结构

            
        注意：
            ！ 必须是英文输入法下的符号


☞ HTML中的单标签：

      1. <meta charset="UTF-8">   作用： 防止网页出现乱码

      2. <br>  ： 实现换行

      3. <hr>  :  表示网页中的一条横线

      4. <!--  内容  --> ： 注释标签，解释说明的意思


☞ HTML中的双标签：
     
     1. 设置网页标题：  <title>  </title>    


         注意： 该标签要放到head标签中


     2. 网页中内容标题：

        <h1></h1>
        <h2></h2>
        <h3></h3>
        <h4></h4>
        <h5></h5>
        <h6></h6>

          注意： 如果我们希望一行表示标题的含义，那么就使用标题标签。


    3. 网页中表示段落的标签：

        <p></p>


    4. 文字格式化标签：

        ✔ 设置文字加粗标签   <strong></strong>(语义)    |   <b> </b>

        ✔ 改变文字大小  
        
            <font></font>

            通过 size属性改变文字大小，不需要加单位

              例如： <font size="10">我是比较大的文字</font>


            通过 color属性改变文字颜色

              例如： <font size="10" color="red">我是比较大的文字</font>


       ✔ 设置文字倾斜：

            <em> </em>  |  <i></i>

       
       ✔ 实现删除线效果

           <del></del>   | <s></s>

       ✔ 设置文字下划线标签：

          <ins></ins>  |  <u></u>


    4. 图片标签：

          ☞ 单标签：  <img>

          ☞重要的属性：  
          
                src【用来设置要显示图片的路径】

                width 【改变标签宽度】

                height 【改变标签高度】

                alt 【当图片无法显示的时候，可以通过alt属性用来描述图片信息】

                title【当鼠标悬停到图片上，对图片的描述信息】


            例如：  <img src="1.jpg" width="500" title="这是一个logo图片" alt="描述信息">

    5. 超链接： 超文本的本质就是将文字放到了超链接标签中

          作用： 实现页面跳转


          标签：
              <a></a>


           属性:
             href: 用来设置要跳转的目标页面路径

             target: 用来设置新页面打开方式

                _self(默认值： 在本窗口中打开页面)

                _blank(在新窗口中打开页面)


            备注：
                1.如果希望本页面中所有的超链接都在新窗口中打开，不需要给每一个a标签添加target属性

                2. 可以通过添加  <base>标签可以实现

                例如：  <base target="_blank">  在head标签中


    6. 锚链接：

         注意：
             1. 锚链接其实也是  a 标签

             2. 当实现从一个页面跳转到另外一个页面的时候，要使用超链接

             3. 如果实现在本页面中跳转，使用锚链接


         锚链接实现方式：
             
             <a href="#"></a>

             在本页面内部跳转
